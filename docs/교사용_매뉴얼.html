🏢 4-2 주식회사

교사용 운영 매뉴얼

앱 소개

4-2 주식회사는 초등학교 교실에서 주식 시뮬레이션을 통해 경제 개념을 배우는 교육용 웹 앱입니다.

각 학생이 자신의 1인 기업(주식회사)을 만들고, 학교생활 성과에 따라 주가가 변동됩니다.
학생들은 가상 화폐로 친구의 회사에 투자하며 경제 원리를 자연스럽게 학습합니다.

시작하기

최초 설정

  1. 앱에 접속합니다.
  2. 관리자 > 학생관리 탭으로 이동합니다.
  3. 일괄 등록 버튼을 클릭합니다.
  4. 학생 이름을 한 줄에 하나씩 입력하고 일괄 등록하기를 누릅니다.
  5. 회사 이름은 자동으로 "OOO 주식회사"로 생성됩니다.
  6. 학생들에게 회사 이름과 슬로건을 직접 정하게 합니다.
"예시 데이터" 버튼으로 8명의 샘플 데이터를 불러와 미리 테스트해볼 수 있습니다.

학생 정보 수정

  • 학생관리 탭에서 회사 이름, 슬로건, 이모지를 클릭하면 바로 수정 가능
  • 학생 추가: 개별 추가 버튼 (전학생 등)
  • 학생 삭제: 해당 학생 옆 휴지통 아이콘

초기 상태

항목값

시작 현금10,000원
주식 보유 한도1개 회사당 최대 3주

주간 운영 흐름

한 주의 흐름 (권장)

월요일 미션 설정 + 지난 주 보고서 나눠주기
화~목 수업 중 포인트 부여 / 행운권 지급
금요일 미션 등급 입력 → 투표 → 정산 실행 → 거래 시간 (주식 매매)

미션 설정 관리자 > 미션

  1. 이번 주 미션을 입력합니다. (예: "독서록 3편 작성하기")
  2. 설정 버튼을 누르면 헤더에 미션이 표시됩니다.
  3. 금요일까지 각 학생에게 미션 등급(S/A/B/미완수)을 부여합니다.
등급주가 반영사업 수익 (현금)
S+500+300원
A+300+200원
B+100+100원
미완수-1000원

선생님 포인트 관리자 > 포인트

수업 태도, 과제, 생활 등을 평가하여 포인트를 줍니다.

  • +100, +50: 좋은 행동
  • -50, -100: 개선 필요
포인트는 정산 시 주가에 직접 반영됩니다.

행운권 관리자 > 행운권

아침활동, 독서록, 돌발미션 등 다양한 사유로 행운권을 지급합니다.

지급 시 학생이 선택:

  • 가상 화폐: 행운권 1장 = 100원으로 즉시 현금 전환
  • 실물 행운권: 실물 행운권으로 받기 (학기말 추첨 등에 사용)
사유행운권 수
🌅 아침활동1장
📖 독서록2장
⭐ 돌발미션 (쉬움)1장
⭐⭐ 돌발미션 (보통)2장
⭐⭐⭐ 돌발미션 (어려움)3장

🎫 기타 (직접 입력)자유 설정

친구 평가/투표 관리자 > 투표

  1. 학생들이 "이번 주 가장 열심히 한 친구"에게 투표합니다.
  2. 선생님이 투표 결과를 입력합니다 (+1표 버튼).

정산 시 반영:

  • 상위 30%: +400
  • 중위: +100
  • 하위 30%: -100
초기화 버튼으로 매주 투표를 리셋합니다.

정산 관리자 > 정산

  1. 포인트, 미션 등급, 투표가 모두 입력되었는지 확인합니다.
  2. 정산 미리보기에서 각 학생의 변동 사항을 확인합니다.
  3. 정산 실행 버튼을 누릅니다.

정산 시 일어나는 일:

  • 포인트 + 미션 + 투표 점수 → 주가에 반영
  • 새 주가의 10% → CEO 급여 (현금 지급)
  • 미션 등급에 따른 → 사업 수익 (현금 지급)
  • 주차가 1 증가
  • 모든 포인트/미션/투표 초기화
정산은 되돌릴 수 없습니다! 미리보기를 꼭 확인하세요.

거래 (거래소 탭)

학생들이 직접 또는 선생님이 대리로 주식을 매매합니다.

매매 방법

  1. 매수/매도 선택
  2. 투자자 선택 (누가 사는지)
  3. 대상 회사 선택 (어느 회사 주식인지)
  4. 수량 선택 (1~3주)
  5. 매수하기/매도하기 클릭

규칙

규칙설명
자기 주식 매수 불가자기 회사 주식은 살 수 없음
보유 한도한 회사당 최대 3주까지 보유 가능
잔액 확인잔액이 부족하면 매수 불가
보유 확인보유하지 않은 주식은 매도 불가

운영 팁

  • 금요일 정산 후 10~15분 거래 시간을 줍니다.
  • 학생들이 "왜 이 회사에 투자하는지" 이유를 말하게 하면 교육 효과 UP
  • 처음엔 선생님이 대리 거래, 익숙해지면 학생 스스로 (태블릿 등)
💬 활용 예시
"김도윤 학생, 왜 아이디어 랩에 투자하고 싶어요?"
"이서연이가 이번 주 미션도 S받고, 발표도 잘해서 주가가 더 오를 것 같아요!"

관리자 기능 상세

특별 이벤트 관리자 > 이벤트

깜짝 호재/악재를 발생시켜 포인트를 줍니다.

  • 대상: 전체 또는 특정 학생 선택
  • 금액: -200 ~ +1000
  • 예시: "청소 우수 보너스 +200!", "지각 -100"

일괄 지급 관리자 > 일괄지급

전체 학생에게 동일한 현금을 지급하거나 차감합니다.

  • 사용 예시: 기본 용돈 지급, 세금 징수, 특별 보너스
  • 금액을 선택하거나 직접 입력
  • 차감 시 잔액 0원 미만으로는 내려가지 않음

시장 조절 관리자 > 시장조절

전체 주가를 비율(%)로 올리거나 내립니다.

  • 호황: 전체 주가 상승 (예: +10%)
  • 불황: 전체 주가 하락 (예: -10%)
  • 미리보기에서 변동 결과를 확인할 수 있음
💬 활용 예시
"오늘 실제로 코스피가 3% 올랐으니, 우리 교실 시장도 호황입니다!"

활동 현황 관리자 > 활동현황

학생별 활동 데이터를 한눈에 확인합니다.

  • 거래 횟수, 행운권 수, 투자 종목 수, 투자자 수
  • 주가 상승률 TOP / 총자산 TOP 랭킹
  • 수업 중 칭찬이나 피드백에 활용

경매 관리자 > 경매

학기말 주주총회에서 과자 경매를 진행할 수 있습니다.

  1. 경매 물품을 등록합니다 (이름, 이모지, 시작가).
  2. 경매 시작 버튼을 누릅니다.
  3. 학생 이름을 클릭하면 100원씩 입찰가가 올라갑니다.
  4. 더 이상 입찰자가 없으면 낙찰 버튼을 누릅니다.
  5. 낙찰가가 해당 학생의 현금에서 차감됩니다.

보고서 관리자 > 보고서

학생 개인별 주간 보고서를 인쇄할 수 있습니다.

  • A4 용지에 1명씩 출력
  • 주가 순위, 자산 현황, CEO 수익, 투자 내역, 주가 추이 포함
  • 학생이 직접 "소감"을 작성하는 란 포함
  • 미리보기/인쇄 버튼으로 바로 출력

수업 운영 팁

📗 첫 주 (도입)

  1. "주식이 뭘까?" 이야기 나누기
  2. 각자 회사 이름, 슬로건 정하기
  3. 이모지로 회사 로고 만들기
  4. 첫 미션 설정 후 1주일 관찰

📘 2~3주차 (적응)

  1. 정산 과정을 학생들과 함께 보기 (빔프로젝터)
  2. 투표 시간에 "왜 이 친구에게 투표하는지" 발표
  3. 거래 시간에 투자 이유 발표하기
  4. 보고서 나눠주고 소감 작성

📙 4주차 이후 (심화)

  1. 시장 조절 기능으로 호황/불황 경험
  2. 뉴스와 연계 (예: "금리 인상 → 불황")
  3. 학생들이 투자 전략 발표하기
  4. 반 경제 신문 만들기

📕 학기말

  1. 과자 경매로 보유 현금 활용
  2. 최종 순위 시상
  3. "나의 경제 활동 돌아보기" 활동
처음에는 선생님이 모든 조작을 하다가, 2~3주차부터 학생들에게 태블릿을 줘서 직접 거래하게 하면 몰입도가 높아집니다.

자주 묻는 질문 (FAQ)

데이터가 날아갔어요!
데이터는 Firebase 클라우드에 자동 저장됩니다. 인터넷 연결을 확인해주세요. 새로고침하면 서버에서 최신 데이터를 불러옵니다.
주가가 너무 높아졌어요/낮아졌어요
시장 조절 기능으로 전체 주가를 비율로 조정할 수 있습니다. 또는 포인트를 조절하여 다음 정산에서 균형을 맞출 수 있습니다.
잘못 거래했어요
반대 거래를 실행하세요. (매수 실수 → 매도로 되돌리기) 직접 수정 기능은 없으므로, 반대 거래로 원상복구합니다.
정산을 잘못 실행했어요
정산은 되돌릴 수 없습니다. 다음 주차에 포인트 조정으로 보정해주세요.
전학생이 왔어요
학생관리 > 개별 추가로 새 학생을 등록합니다. 시작 주가 1,000원, 현금 10,000원으로 시작합니다. 필요하면 일괄지급으로 추가 현금을 줄 수 있습니다.
학생이 전학갔어요
학생관리에서 해당 학생을 삭제합니다. 해당 학생에게 투자한 다른 학생의 주식은 자동으로 사라지므로, 삭제 전에 관련 학생들의 매도를 먼저 처리하는 것을 권장합니다.
모바일에서도 사용할 수 있나요?
네! 스마트폰, 태블릿에서도 사용 가능합니다. 모바일에서는 하단 탭바로 메뉴를 이동합니다.

정산 계산식 상세

주가 계산

새 주가 = MAX(100, 현재 주가 + 선생님 포인트 + 미션 점수 + 투표 점수)

주가는 최소 100원 이하로 내려가지 않습니다.

CEO 급여

CEO 급여 = 새 주가 × 10% (소수점 반올림)

정산 후 새 주가 기준으로 계산됩니다.

사업 수익

미션 등급사업 수익
S+300원
A+200원
B+100원
미완수0원

최종 현금

새 현금 = 현재 현금 + CEO 급여 + 사업 수익

예시 계산

현재 주가 1,500원 / 포인트 +100 / 미션 A(+300) / 투표 상위(+400)

새 주가 = 1,500 + 100 + 300 + 400 = 2,300원 CEO 급여 = 2,300 × 10% = 230원 사업 수익 = 200원 (A등급) 현금 증가 = 230 + 200 = 430원

국어회사 소개 글쓰기, 투자 전략 발표, 경제 신문 만들기
