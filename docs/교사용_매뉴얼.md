# 4-2 주식회사 - 교사용 운영 매뉴얼

## 1. 앱 소개

**4-2 주식회사**는 초등학교 교실에서 주식 시뮬레이션을 통해 경제 개념을 배우는 교육용 웹 앱입니다.

각 학생이 자신의 1인 기업(주식회사)을 만들고, 학교생활 성과에 따라 주가가 변동됩니다.
학생들은 가상 화폐로 친구의 회사에 투자하며 경제 원리를 자연스럽게 학습합니다.

### 핵심 개념
| 개념 | 설명 |
|------|------|
| 주식회사 | 학생 1명 = 1개 회사. 회사 이름, 슬로건, 이모지를 직접 정함 |
| 주가 | 학교생활 성과(포인트, 미션, 투표)에 따라 매주 변동 |
| 현금 | CEO 급여 + 사업 수익 + 행운권으로 얻음. 투자에 사용 |
| 투자 | 다른 친구의 회사 주식을 사고팔 수 있음 (최대 3주) |
| 정산 | 매주 1회 실행. 포인트/미션/투표 → 주가 반영, 급여 지급 |

---

## 2. 시작하기

### 2-1. 최초 설정
1. 앱에 접속합니다.
2. **관리자 > 학생관리** 탭으로 이동합니다.
3. **일괄 등록** 버튼을 클릭합니다.
4. 학생 이름을 한 줄에 하나씩 입력하고 **일괄 등록하기**를 누릅니다.
5. 회사 이름은 자동으로 "OOO 주식회사"로 생성됩니다.
6. 학생들에게 회사 이름과 슬로건을 직접 정하게 합니다.

> **팁**: "예시 데이터" 버튼으로 8명의 샘플 데이터를 불러와 미리 테스트해볼 수 있습니다.

### 2-2. 학생 정보 수정
- **학생관리** 탭에서 회사 이름, 슬로건, 이모지를 클릭하면 바로 수정 가능합니다.
- 학생 추가: **개별 추가** 버튼 (전학생 등)
- 학생 삭제: 해당 학생 옆 휴지통 아이콘

### 2-3. 초기 상태
- 모든 학생의 시작 주가: **1,000원**
- 모든 학생의 시작 현금: **10,000원**
- 주식 보유 한도: 1개 회사당 **최대 3주**

## 3. 주간 운영 흐름

### 한 주의 흐름 (권장)

```
월요일: 미션 설정 + 지난 주 보고서 나눠주기
화~목:  수업 중 포인트 부여 / 행운권 지급
금요일: 미션 등급 입력 → 투표 → 정산 실행
       → 거래 시간 (주식 매매)
```

### 3-1. 미션 설정 (관리자 > 미션)
1. 이번 주 미션을 입력합니다. (예: "독서록 3편 작성하기")
2. **설정** 버튼을 누르면 헤더에 미션이 표시됩니다.
3. 금요일까지 각 학생에게 미션 등급(S/A/B/미완수)을 부여합니다.

| 등급 | 주가 반영 | 사업 수익 (현금) |
|------|-----------|-----------------|
| S | +500 | +300원 |
| A | +300 | +200원 |
| B | +100 | +100원 |
| 미완수 | -100 | 0원 |

### 3-2. 선생님 포인트 (관리자 > 포인트)
수업 태도, 과제, 생활 등을 평가하여 포인트를 줍니다.
- **+100, +50**: 좋은 행동
- **-50, -100**: 개선 필요

> 포인트는 정산 시 주가에 직접 반영됩니다.

### 3-3. 행운권 (관리자 > 행운권)
아침활동, 독서록, 돌발미션 등 다양한 사유로 행운권을 지급합니다.

지급 시 학생이 선택:
- **가상 화폐**: 행운권 1장 = 100원으로 즉시 현금 전환
- **실물 행운권**: 실물 행운권으로 받기 (학기말 추첨 등에 사용)

| 사유 | 행운권 수 |
|------|-----------|
| 아침활동 | 1장 |
| 독서록 | 2장 |
| 돌발미션 (쉬움) | 1장 |
| 돌발미션 (보통) | 2장 |
| 돌발미션 (어려움) | 3장 |
| 돌발미션 (최고난도) | 5장 |
| 기타 (직접 입력) | 자유 설정 |

### 3-4. 친구 평가/투표 (관리자 > 투표)
1. 학생들이 "이번 주 가장 열심히 한 친구"에게 투표합니다.
2. 선생님이 투표 결과를 입력합니다 (+1표 버튼).
3. 정산 시 반영:
   - 상위 30%: **+400**
   - 중위: **+100**
   - 하위 30%: **-100**

> **초기화** 버튼으로 매주 투표를 리셋합니다.

### 3-5. 정산 (관리자 > 정산)
1. 포인트, 미션 등급, 투표가 모두 입력되었는지 확인합니다.
2. **정산 미리보기**에서 각 학생의 변동 사항을 확인합니다.
3. **정산 실행** 버튼을 누릅니다.

정산 시 일어나는 일:
- 포인트 + 미션 + 투표 점수 → **주가에 반영**
- 새 주가의 10% → **CEO 급여** (현금 지급)
- 미션 등급에 따른 → **사업 수익** (현금 지급)
- 주차가 1 증가
- 모든 포인트/미션/투표 초기화

> **주의**: 정산은 되돌릴 수 없습니다! 미리보기를 꼭 확인하세요.

## 4. 거래 (거래소 탭)

학생들이 직접 또는 선생님이 대리로 주식을 매매합니다.

### 매매 방법
1. **매수/매도** 선택
2. **투자자** 선택 (누가 사는지)
3. **대상 회사** 선택 (어느 회사 주식인지)
4. **수량** 선택 (1~3주)
5. **매수하기/매도하기** 클릭

### 규칙
- 자기 회사 주식은 살 수 없음
- 한 회사당 최대 3주까지 보유 가능
- 잔액이 부족하면 매수 불가
- 보유하지 않은 주식은 매도 불가

### 운영 팁
- 금요일 정산 후 **10~15분 거래 시간**을 줍니다.
- 학생들이 "왜 이 회사에 투자하는지" 이유를 말하게 하면 교육 효과 UP
- 처음엔 선생님이 대리 거래, 익숙해지면 학생 스스로 (태블릿 등)

## 5. 관리자 기능 상세

### 5-1. 특별 이벤트 (관리자 > 이벤트)
깜짝 호재/악재를 발생시켜 포인트를 줍니다.
- 대상: 전체 또는 특정 학생 선택
- 금액: -200 ~ +1000
- 예시: "청소 우수 보너스 +200!", "지각 -100"

### 5-2. 일괄 지급 (관리자 > 일괄지급)
전체 학생에게 동일한 현금을 지급하거나 차감합니다.
- 사용 예시: 기본 용돈 지급, 세금 징수, 특별 보너스
- 금액을 선택하거나 직접 입력
- 차감 시 잔액 0원 미만으로는 내려가지 않음

### 5-3. 시장 조절 (관리자 > 시장조절)
전체 주가를 비율(%)로 올리거나 내립니다.
- **호황**: 전체 주가 상승 (예: +10%)
- **불황**: 전체 주가 하락 (예: -10%)
- 미리보기에서 변동 결과를 확인할 수 있음
- 수업에서 경제 뉴스와 연계하여 사용하면 효과적

> 활용 예: "오늘 실제로 코스피가 3% 올랐으니, 우리 교실 시장도 호황입니다!"

### 5-4. 활동 현황 (관리자 > 활동현황)
학생별 활동 데이터를 한눈에 확인합니다.
- 거래 횟수, 행운권 수, 투자 종목 수, 투자자 수
- 주가 상승률 TOP / 총자산 TOP 랭킹
- 수업 중 칭찬이나 피드백에 활용

### 5-5. 경매 (관리자 > 경매)
학기말 주주총회에서 과자 경매를 진행할 수 있습니다.
1. 경매 물품을 등록합니다 (이름, 이모지, 시작가).
2. **경매 시작** 버튼을 누릅니다.
3. 학생 이름을 클릭하면 100원씩 입찰가가 올라갑니다.
4. 더 이상 입찰자가 없으면 **낙찰** 버튼을 누릅니다.
5. 낙찰가가 해당 학생의 현금에서 차감됩니다.

### 5-6. 보고서 (관리자 > 보고서)
학생 개인별 주간 보고서를 인쇄할 수 있습니다.
- A4 용지에 1명씩 출력
- 주가 순위, 자산 현황, CEO 수익, 투자 내역, 주가 추이 포함
- 학생이 직접 "소감"을 작성하는 란 포함
- **미리보기/인쇄** 버튼으로 바로 출력

## 6. 화면 설명

### 대시보드
- 상장 기업 수, 평균 주가, 총 거래, 현재 주차 통계
- 전체 주가 추이 그래프
- 종목별 카드 (클릭하면 상세 정보)

### 학생 상세 (종목 카드 클릭)
- 현재 주가, 보유 현금, 총 자산, 누적 수익률
- 이번 주 변동, CEO 급여, 사업 수익
- 주가 추이 차트
- 우리 회사 투자자 (누가 나에게 투자했는지)
- 보유 주식 (투자 원금/현재 가치/손익)
- 최근 거래 내역

### 거래소
- 주식 매매 폼 (매수/매도)
- 거래 기록 (최근 30건)

### 순위표
- 주가 순위 / 총자산 순위 전환
- 1~3위 포디움 + 나머지 리스트
- 비교 차트

## 7. 수업 운영 팁

### 첫 주 (도입)
1. "주식이 뭘까?" 이야기 나누기
2. 각자 회사 이름, 슬로건 정하기
3. 이모지로 회사 로고 만들기
4. 첫 미션 설정 후 1주일 관찰

### 2~3주차 (적응)
1. 정산 과정을 학생들과 함께 보기 (빔프로젝터)
2. 투표 시간에 "왜 이 친구에게 투표하는지" 발표
3. 거래 시간에 투자 이유 발표하기
4. 보고서 나눠주고 소감 작성

### 4주차 이후 (심화)
1. 시장 조절 기능으로 호황/불황 경험
2. 뉴스와 연계 (예: "금리 인상 → 불황")
3. 학생들이 투자 전략 발표하기
4. 반 경제 신문 만들기

### 학기말
1. 과자 경매로 보유 현금 활용
2. 최종 순위 시상
3. "나의 경제 활동 돌아보기" 활동

## 8. 자주 묻는 질문 (FAQ)

### Q. 데이터가 날아갔어요!
A. 데이터는 Firebase 클라우드에 자동 저장됩니다. 인터넷 연결을 확인해주세요.
새로고침하면 서버에서 최신 데이터를 불러옵니다.

### Q. 주가가 너무 높아졌어요/낮아졌어요
A. **시장 조절** 기능으로 전체 주가를 비율로 조정할 수 있습니다.
또는 포인트를 조절하여 다음 정산에서 균형을 맞출 수 있습니다.

### Q. 잘못 거래했어요
A. 반대 거래를 실행하세요. (매수 실수 → 매도로 되돌리기)
직접 수정 기능은 없으므로, 반대 거래로 원상복구합니다.

### Q. 정산을 잘못 실행했어요
A. 정산은 되돌릴 수 없습니다. 다음 주차에 포인트 조정으로 보정해주세요.

### Q. 전학생이 왔어요
A. **학생관리 > 개별 추가**로 새 학생을 등록합니다.
시작 주가 1,000원, 현금 10,000원으로 시작합니다.
필요하면 **일괄지급**으로 추가 현금을 줄 수 있습니다.

### Q. 학생이 전학갔어요
A. **학생관리**에서 해당 학생을 삭제합니다.
해당 학생에게 투자한 다른 학생의 주식은 자동으로 사라지므로,
삭제 전에 관련 학생들의 매도를 먼저 처리하는 것을 권장합니다.

### Q. 모바일에서도 사용할 수 있나요?
A. 네! 스마트폰, 태블릿에서도 사용 가능합니다.
모바일에서는 하단 탭바로 메뉴를 이동합니다.

## 9. 정산 계산식 상세

```
새 주가 = MAX(100, 현재 주가 + 선생님 포인트 + 미션 점수 + 투표 점수)

CEO 급여 = 새 주가 × 10%  (소수점 반올림)

사업 수익 = 미션 등급별 (S: 300원, A: 200원, B: 100원, 미완수: 0원)

새 현금 = 현재 현금 + CEO 급여 + 사업 수익
```

> 주가는 최소 100원 이하로 내려가지 않습니다.

## 10. 교육과정 연계

| 교과 | 연계 활동 |
|------|-----------|
| 사회 | 경제 개념 (생산, 소비, 투자, 시장), 금융 교육 |
| 수학 | 퍼센트 계산, 손익 계산, 그래프 읽기 |
| 국어 | 회사 소개 글쓰기, 투자 전략 발표, 경제 신문 만들기 |
| 도덕 | 공정한 경쟁, 협력, 정직한 거래 |
| 창체 | 진로 교육 (CEO 체험), 경제 교육 |
